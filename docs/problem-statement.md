# Problem Statement

Metabolomics analysis for placenta and cord blood data is complex, multi-step, and difficult to standardize when done manually.

The project needs a reproducible workflow that:

- Organizes raw and processed datasets consistently.
- Produces interpretable PCA and outlier analysis outputs.
- Supports downstream biomarker modeling and reporting.
- Enables clear, continuous documentation for team handoff and future updates.

This repository addresses those needs with a structured notebook pipeline and organized outputs.
