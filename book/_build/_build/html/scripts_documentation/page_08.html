Page 8: ML Modeling (Cord GDM)#

Source script:

  • src/scripts/08_ML_modeling.ipynb

Purpose#

Runs metabolite-level statistical modeling for cord data against GDM outcome and exports ranked results.

What This Script Does#

  1. Loads cord metabolite data and GDM mapping information.

  2. Fits metabolite-wise statistical models against GDM status.

  3. Computes raw and corrected p-values (including q-values).

  4. Calculates group-level summary statistics (for example mean in GDM vs non-GDM).

  5. Exports a ranked results table for downstream filtering and interpretation.
