Page 4: PCA Analysis#

Source script:

  • src/scripts/04_pca_analysis.ipynb

Purpose#

Runs PCA for placenta-only, cord-only, and combined datasets, and saves PCA scores used downstream.

What This Script Does#

  1. Loads processed placenta and cord metabolite matrices.

  2. Runs PCA separately for placenta and cord datasets.

  3. Runs PCA on the combined placenta + cord dataset.

  4. Saves PCA score tables for all three analyses.

  5. Generates PCA plots (for example scree and PC1 vs PC2) used in later interpretation.
