# Dataset 2: MetaPro_cord_blood_Jan2022.xlsx

`MetaPro_cord_blood_Jan2022.xlsx` contains metabolomics measurements from human umbilical cord blood. It reflects the newborn metabolic profile at delivery and can be compared with placenta metabolomics.

## Context

This dataset helps characterize biochemical differences in fetal-side biology and supports cross-tissue comparisons under the same study design.

## Samples

- Total samples: 38
- Sample labels: `sample-1` to `sample-38`
- Experimental groups: Group A and Group B

## Metabolites

Each row is one metabolite measured in cord blood, including classes such as:

- Amino acids
- Lipids
- Vitamins and cofactors

## Columns

- Identifier fields: `HMDB ID`, `PUBCHEM ID`, `KEGG ID`, and related annotations
- Acquisition method fields (for example `NEGa`, `POSb`)
- Sample value columns: `sample-1` to `sample-38`

## Workbook sheets

- Sample Info
- Peak Area Data
- Normalized Data
- Data Key

In short, this file provides chemical profiles for 38 cord blood samples to compare metabolite/pathway differences across study groups.
